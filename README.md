# vexis-app

 **status:** fork-active
 **origin:** [SimpleBluetoothLeTerminal](https://github.com/kai-morich/SimpleBluetoothLeTerminal.git)
 **intent:** hardware-voice-override

---

### the modification
this is a specialized interface for **vexis**. 

the original terminal code has been modified to include **voice functionality**. you can now control the hardware through spoken commands instead of manual input.

---

### quick-start
1. **clone** the repo.
2. **build** for android.
3. **link** with vexis hardware.
